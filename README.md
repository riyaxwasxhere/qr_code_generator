# 🔳 QR Code Generator

A simple QR Code Generator built using HTML, CSS, and JavaScript. It allows users to generate QR codes instantly from any text or URL, making it easy to share information in a scannable format.

---

## ✨ Features

- 🔗 Generate QR codes from text or URLs
- ⚡ Instant QR code generation
- 📱 Responsive user interface
- 🎨 Clean and simple design
- 📷 Download or scan generated QR codes (if implemented)

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- QR Code API / Library

---

## 🚀 Getting Started

```bash
git clone https://github.com/riyaxwasxhere/qr_code_generator.git
```

Open `index.html` in your browser.
